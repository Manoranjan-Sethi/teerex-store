## Tech Stack

- HTML 5
- CSS 3
- Javascript
- React JS
- Context API + useReducer

## Features

- Product Page
- Filter
- Add to cart

## Run Locally

Clone the project

```bash
  git clone https://github.com/Manoranjan-Sethi/teerex-store.git
```

Go to folder

```bash
 cd teerex-store
```

Install packages

```bash
 npm i
```

Run the app

```bash
  npm start
```
